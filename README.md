# 🧨 Minesweeper Game

A command-line implementation of the classic **Minesweeper** game built with **Python**. The game features random bomb placement, recursive cell revealing, input validation, and dynamic board rendering.

## 🚀 Python Concepts Used

* Object-Oriented Programming (OOP)
* Classes & Objects
* Recursion
* 2D Lists & Sets
* Loops & Conditional Statements
* List Comprehensions
* Regular Expressions (`re`)
* Modules (`random`, `re`)
* String Formatting (f-strings)

## ▶️ Run

```bash
python minesweeper.py
```
